Instalasi ArchLinux di PC/Laptop

Persiapan: 1. PC/Laptop 64bit. 2. RAM minimum 512MB. (rekomendasi 2GB) 3. Minimal 16GB HDD/SSD. 4. Koneksi internet. 5. USB flashdisk 4GB. 6. Paham beberapa perintah dasar linux. Langkah 1: Download File Instalasi (ISO) Arch Linux Download dan simpan ISO Arch Linux dari website resminya. Langkah 2: Burn/Install File ISO Ke USB Gunakan Rufus untuk membuat instalasi live medium ISO Arch Linux ke USB flashdisk. Langkah 3: Booting Live USB Arch Linux Gunakan USB flashdisk yang telah di install/burn ISO Arch Linux untuk booting di PC/Laptop. Setelah itu pilih opsi "Boot Arch Linux (x86_64), tunggu hingga anda masuk ke sistem dengan mode root.
